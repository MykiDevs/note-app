# Simple REST API note management backend app
### Prerequisites
- Docker
- Java 25

### Running with Docker
1. Copy .env.example to .env and customize environment variables if needed:
```bash
cp .env.example .env
```
2. **Start the application**
```bash
docker compose up --build
```
## API Reference
http://localhost:8080/swagger-ui/index.html`
